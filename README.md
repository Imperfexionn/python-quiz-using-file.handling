A simple Python quiz application that asks a set of questions, records your answers, and tracks your score.

### Features

* Stores questions and answers in a list (`quiz`), making it easy to add or modify them.
* Prompts the user for answers and compares them to the correct answers.
* Saves results to a `scores.txt` file.
* Displays the results in the console after finishing the quiz.

### How to Use

1. Make sure Python is installed on your computer.
2. Run the script.
3. Enjoy.
